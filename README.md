# ☠️ Как скачать?

*- 1.Установите билд 1387* ***CCode***
``Пока его можно скачать в гугл плей.``

*-Дальше скачайте этот репозиторий нажав на кнопку* ***Code*** *а потом* ***Download ZIP*** *дальше распакуйте эту папку и удалите всё кроме ccode файла.*

*-Заходим в CCode и нажмием на кнопку*
***Мои проекты*** а потом и ***+*** *и там выбираем наш проект* ***готово!***
